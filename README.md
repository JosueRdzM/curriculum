# 📄 Mi Currículum Vitae Web — Josué Rodríguez

¡Bienvenido a mi repositorio central de trayectoria profesional! Este proyecto alberga mi currículum operativo y tecnológico en un formato web responsivo y de código abierto.

Puedes ver la versión en vivo aquí: [josuerdzm.github.io/checklistayv.github.io](https://josuerdzm.github.io/checklistayv.github.io)

---

## 🎯 La Necesidad

En el entorno tecnológico y administrativo actual, un currículum estático en formato PDF o papel resulta limitado. Como profesional en formación en la Ingeniería en Sistemas Computacionales y especialista en optimización operativa, surge la necesidad obvia de predicar con el ejemplo: **centralizar, documentar y albergar mi historia laboral en un espacio digital propio, accesible y dinámico.** 

Este repositorio no solo expone mis habilidades y experiencia de manera pública para reclutadores y colaboradores, sino que también funciona como un portafolio vivo de buenas prácticas de desarrollo web, control de versiones y accesibilidad.

---

## ⚡ Desarrollo con Vibe Coding

Este sitio fue concebido y desarrollado bajo el enfoque moderno de **Vibe Coding**. En lugar de quedar atrapado en la sintaxis repetitiva, utilicé Inteligencia Artificial generativa como copiloto estratégico para iterar rápidamente la estructura en HTML5 y los estilos CSS3. 

Esto me permitió:
- **Agilidad extrema:** Diseñar una interfaz limpia, elegante y completamente responsiva en una fracción del tiempo habitual.
- **Enfoque en la arquitectura:** Centrar mis esfuerzos en la jerarquía de la información, la lógica del diseño y la experiencia de usuario (UX), delegando la escritura del código base a herramientas de IA bajo mi supervisión técnica.
- **Estructura modular:** Mantener estilos CSS limpios y variables nativas para facilitar futuros cambios de diseño en segundos.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructuración semántica de la información laboral y proyectos.
- **CSS3:** Diseño personalizado, variables nativas (`:root`) para la paleta de colores y *Media Queries* para adaptabilidad móvil.
- **GitHub Pages:** Infraestructura de despliegue continuo (CD) para albergar el sitio de forma gratuita y eficiente.
